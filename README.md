# SSHConnector
## 项目简介
### 声明
本项目遵循 GPL v3 开源协议
这是一个由C++开发的SSH客户端项目
在此项目中，使用了[proxysocket](https://github.com/brechtsanders/proxysocket/releases/tag/0.1.11)开源项目的代码
### 特点
+ 静态编译后文件大小仅350kb，极其简洁
+ 支持使用SOCKS4/SOCKS5代理服务器
+ 使用UTF-8编码，防止乱码问题
+ 使用pty xterm，支持使用top等指令
